nautilusexif
============

Nautilus context menu for metadata (EXIF, IPTC, XMP) manipulation.

Install
-------
Ensure exiftool is installed:

    apt-get install exiftool

and place the script (`tag`) in `~/gnome2/nautilus-scripts`.
